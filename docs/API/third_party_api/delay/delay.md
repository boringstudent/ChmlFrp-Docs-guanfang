# 获取所有节点的域名
- 接口
    > GET /delay
- 请求参数

    **JSON:**

    |  参数   |  类型  | 必传 |       说明        |
    |:-----:|:----:|:--:|:---------------:|
    | token | str  | 是  | 用于验证是否返回vip节点信息 |

- 返回

    **JSON:**

    |  参数   |  类型  | 必返 |  说明  |
    |:-----:|:----:|:--:|:----:|
    |  msg  | str  | 是  | 返回消息 |
    | data  | list | 是  | 详细如下 |

    **data:**

    |    参数     | 类型  | 必返 |  说明  |
    |:---------:|:---:|:--:|:----:|
    | node_name | str | 是  | 节点名称 |
    |    ip     | str | 是  | 节点域名 |
